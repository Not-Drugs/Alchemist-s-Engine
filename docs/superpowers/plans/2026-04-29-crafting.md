# Crafting System v1 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship Minecraft-style pattern crafting on the existing 6×4 alchemy grid, plus the first recipe (Stick Golem). Players drag raw mats from a new Inventory rail, arrange them in a `+` around a spark, tap a contextual Craft button, and the resulting golem lands in a Key Items Bag.

**Architecture:** Three new state buckets (`game.inventory`, `game.satchel`, `game.keyItems`) added to the existing single-file game state. A new `ingredient` tile kind on the grid (alongside existing `fuel`/`ore`). A pure pattern-matcher (`findRecipeMatch`) runs after every grid mutation and surfaces a contextual `[Craft]` button when a recipe pattern is present. Two new draggable rails above the grid (Inventory + Satchel), one accordion panel for discovered recipes, one modal for the Key Items Bag. Existing reads/writes of `game.resources.sticks/.stones` migrate to `game.inventory.sticks/.stones`.

**Tech Stack:** Vanilla HTML/CSS/JavaScript, no build step. localStorage save/load. Same drag stack as today (HTML5 drag for desktop + parallel touch implementation). No new dependencies.

**No test infra:** This codebase has zero automated tests by design. Each task ends with a **manual browser verification** step before commit, mirroring the design spec's verification approach.

**Spec reference:** `docs/superpowers/specs/2026-04-29-crafting-design.md`

const KEYITEMS_CAP = 8;
const GRID_W = 6;
const GRID_H = 4;

const RECIPES = [
  {
    id: 'stickGolem',
    name: 'Stick Golem',
    output: { type: 'golem' },
    pattern: {
      shape: 'plus',
      center: { type: 'fuel', tier: 1 },
      arms: { type: 'ingredient', kind: 'stick' },
    },
  },
];

function matchesSpec(cell, spec) {
  if (!cell || !spec) return false;
  if (cell.type !== spec.type) return false;
  if (spec.tier !== undefined && cell.tier !== spec.tier) return false;
  if (spec.kind !== undefined && cell.kind !== spec.kind) return false;
  return true;
}

function matchPlus(grid, centerSpec, armSpec) {
  for (let y = 1; y < GRID_H - 1; y++) {
    for (let x = 1; x < GRID_W - 1; x++) {
      const ci = y * GRID_W + x;
      if (!matchesSpec(grid[ci], centerSpec)) continue;
      const arms = [
        (y - 1) * GRID_W + x,
        (y + 1) * GRID_W + x,
        y * GRID_W + (x - 1),
        y * GRID_W + (x + 1),
      ];
      if (arms.every(i => matchesSpec(grid[i], armSpec))) {
        return [ci, ...arms];
      }
    }
  }
  return null;
}

function findRecipeMatch(grid) {
  for (const recipe of RECIPES) {
    if (recipe.pattern.shape === 'plus') {
      const cells = matchPlus(grid, recipe.pattern.center, recipe.pattern.arms);
      if (cells) return { recipe, cells };
    }
  }
  return null;
}

if (keyItemsBtn && modal) {

}

  });
}
